# Privacy & Compliance Docs

本資料夾包含 Google Play 上架需要的隱私與合規文件（v1）。

- privacy-policy-zh-tw.md：隱私權政策（繁中）
- data-safety-mapping.md：Play Console Data Safety 填寫對照
- in-app-disclosure.md：App 內「照片上傳給第三方 AI」揭露文案
- data-deletion-policy.md：資料刪除政策與流程（含 3 天 / 32 天 retention）

Retention Policy（系統已實作）：
- DRAFT/PENDING/FAILED：3 天後自動刪除（照片 + 結果）
- SAVED：32 天後自動刪除（照片 + 結果 + 歷史）
